# 📰 NewsPulse AI Trend Analyzer

## 📌 Overview
This project analyzes news articles using NLP techniques like sentiment analysis and topic modeling.

## 🚀 Features
- Sentiment Analysis (TextBlob)
- Topic Modeling (LDA)
- News Visualization Dashboard
- Interactive UI using Streamlit

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- NLTK, TextBlob
- Scikit-learn
- Streamlit
- Matplotlib, Plotly

## ⚙️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📂 Dataset
- Multi-genre news dataset used for analysis
